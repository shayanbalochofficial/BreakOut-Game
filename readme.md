# Neon Breakout

A premium arcade-style Breakout game with a neon theme, built using HTML, CSS, and JavaScript. Smash glowing bricks with a bouncing ball, rack up high scores, and enjoy retro vibes with modern glow effects!

## Features
- **Neon Arcade Theme**: Dark cosmic background with vibrant purple, pink, cyan, and lime glows.
- **Responsive Design**: Play seamlessly on desktop, tablet, or mobile with touch/mouse/keyboard controls.
- **Dynamic Gameplay**: Progressive levels, ball trails, particle explosions, and sound effects.
- **High Score Tracking**: Persistent local storage for your best scores.
- **Glow Effects & Animations**: Shimmering buttons, particle bursts, and UI glows for a premium feel.

## How to Play
- **Controls**:
  - Desktop: Arrow keys or mouse drag.
  - Mobile: Touch drag anywhere on the screen.
- **Objective**: Use the paddle to bounce the ball and destroy all bricks. Clear levels to increase speed and score!
- **Rules**: Missing the ball resets the score and level. Aim for high scores!

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/shayanbalochofficial/BreakOut-Game.git
   ```
2. Open `index.html` in any modern web browser (no server needed).

## Tech Stack
- HTML5 Canvas for rendering.
- CSS3 for neon gradients and animations.
- Vanilla JavaScript for game logic, physics, and audio.
